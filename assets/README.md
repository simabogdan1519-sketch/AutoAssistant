# Assets

## Fonturi

Fonturile sunt încărcate prin pachetele `@expo-google-fonts/*` din `package.json` — nu ai nimic de descărcat manual. `assets/fonts/` rămâne gol (cu `.gitkeep` pentru git).

## Imagini

Imaginile placeholder (icon, splash, favicon) sunt deja generate. Pentru publicarea pe Play Store ar trebui să le înlocuiești cu design-ul tău real:

- `icon.png` — 1024×1024, icon-ul principal al aplicației
- `adaptive-icon.png` — 1024×1024, foreground pentru Android adaptive icon
- `splash.png` — 1284×2778, ecran splash
- `favicon.png` — 48×48 pentru versiunea web

**Paletă AutoAssistant:** bg `#0a0a0a`, accent `#d4ff3a`, text `#f5f5f2`.

Poți folosi Figma, Canva sau orice alt tool.
